# Mon CV Éco-responsable

Ce projet est un CV en ligne développé en **Next.js** avec une attention particulière portée à l'**éco-conception**. Il utilise le déploiement statique (`next export`), le Vanilla CSS (via l'optimisation maximale de Tailwind) et un ensemble de bonnes pratiques pour limiter notre empreinte numérique.

## Technologies Utilisées
- Next.js (React)
- Tailwind CSS (purgé)
- React Icons

## Lancement en local

1. Installer les dépendances :
   ```bash
   npm install
   ```

2. Développer en local :
   ```bash
   npm run dev
   ```

3. Compiler pour la production (Génération statique) :
   ```bash
   npm run build
   ```

## Hébergement statique
Le projet est configuré pour générer un dossier `out/` hébergeable gratuitement sur GitHub Pages.
